A sub-Riemannian metric assigns a length to *some* tangent directions only —
the rest are forbidden, but reachable through chained admissible motions.
Geodesics on $\mathrm{SE}(2)$, optimal-control problems with nonholonomic
constraints, and Petitot's V1 model all live in this setting.
